What are some internet and web definitions?

1. Internet

The internet is defined as a global network that connects devices like phones, computers
and others so that information can be shared faster and more efficiently. It is akin to that
of a "highway" that makes things like email, online games, video calls and websites possible.

2. Web (World Wide Web or WWW)

The web is a collection of websites, web pages, videos and images that can be seen by the
users through browser like Google Chrome and Safari. It works through the internet to deliver
information to the users.

3. How are the Internet and Web different?

The internet is a physical network, like the roads, cables, and systems tha connects or
links everything together. Meanwhile, the web is just one service that uses the internet,
like cars driving on roads. The internet is bigger than the web. Thus, without the internet,
the web wouldn't exist.

4. ASCII and Unicode

ASCII is an older system that only has 128 possible characters, which represents basic English
letters, numbers, and symbols. Unicode on the other hand, is a much more modern system that
could represent almost every character in every language, including emojis and symbols,
making this system much more advanced, powerful, and universal.

5. UTF-8

UTF-8 is a method of storing Unicode characters that are using between 1 and 4 bytes. It is
said to be smart because English letters usually take like up to 1 byte (For example, ASCII).
It can also handle thousands of symbols and languages. That is the reason why UTF-8 is the
most commonly used text encoding on the web.

6. URL (Uniform Resource Locator)

A URL is simply the address of something on the web. For example, https://en.wikipedia.org/wiki
/Five_Nights_at_Freddy%27s is the address that takes the user to the Five Nights At Freddy's
Wikipedia website. Without the URL, the computer wouldn't where to go.

7. Domain Name

A domain name is human-friendly version of an address. Instead of typing a long number like
172.217.14.206, we can just type the domain name, which is google.com. The domain name makes
it easier for people to access websites.

8. HTTP (Hypertext Transfer Protocol)

HTTP is a set of rules that lets a user's web browser and a website's server talk to each other
When the user clicks on a link, the HTTP will tell the user's browser how to request the page
and how to display it once received.

9. HTTPS (Hypertext Transfer Protocol Secure)

HTTPS is the secure version of HTTP. Th reason is because it adds encryption so that the user's
personal data and information are safe from anyone who wants to spy or steal the personal
information the user sends such as credit card details and passwords. That is why modern
websites almost always use HTTPS.

10. IP Address (Internet Protocol Address)

An IP address is like the "home address" of a device on a network. Every computer or phone needs
an IP address so that data knows where to go. For example, when the user visits a website, their
computer uses the website's IP address to connect to it.

11. DNS (Domain Name System)

The way the DNS works is like a phonebook for the internet. When the user types in a domain name
like facebook.com, DNS will then translate it into an IP address of the computer where the website
is stored. This way, the user doesn't have to remember long numbers.

12. W3C Recommendations

The W3C or the World Wide Web Consortium is a group that comes up with rules or standards for how the
web should work. Their recommendations make sure that websites have the same appearance and behaviour
no matter what browser or device the user uses

13. XML, XHTML, HTML, HTML5

- XML is a type of language that stores and share data in a clean and structured way.
- XHTML is just HTML but written in a stricter way that follows the XML rules.
- HTML is the language the user uses to build websites by organizing text, images,
appearance and links
- HTML5 is the latest version of HTML. It supports modern features like videos, and
interactive content directly in the browser without extra software.
