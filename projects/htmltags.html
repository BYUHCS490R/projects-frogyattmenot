HTML Tags

1. Line Break (< br >)

The HTML < br > tag is used create a line break within a block of text.
It allows the user to create a new line of text without having to make
a new paragraph.

2. Horizontal Rule (< hr >)

The HTML < hr > tag is used to create a line on the webpage. It allows
the user to visually divide the webpage into sections, making it more
organized.

3. Blockquote (< blockquote >)

The HTML < blockquote > is used to define a section of text that is
quoted from another source. It allows the user to distinguish the quote
from the rest of the content.

4. Phrase Elements

- Emphasis tag (< em >< /em >) is the tag that emphasizes or stresses the texts.

- Mark tag (< mark >< /mark >) defines text with a higlighted yellow background.

- Strong tag (< strong>< /strong> ) bolds the appearance of the text.

- Abbreviation tag (< abbr >< /abbr >) is used to abbreviate any word.

- Acronym tag (< acronym >< /acronym >) is used to show that the text is an acronym.
this doesn't exist in HTML 5.

- Special terminology/Definition tag (< dfn >< /dfn >) is used when a special term is introduced and
is displayed differently.

- Quotation tag (< blockquote >< /blockquote >) is used to add quote to a passage from a different source.

- Short quote tag (< q >< /q >) is used to add double quotation marks to a statement.

- Code tag (< code >< /code ) is used to make the appearance of the text to be mono-spaced font.

- The Keyboard tag (< kbd >< /kbd >) defines text as input from a keyboard.

- The Address tag (< address >< /address >) used to show that a set of text is an address.
