2. What are the two main differences between CSS Grid and CSS Flex?

CSS grid refers to to a grid-based layout with rows and columns. It allows developers to create a complex web layout. It also makes it easy for developers to create a responsive layout that is adaptable without having to use float or positioning. Flex is way easier to do. Grid is two dimensional and is more strict like the parent is commanding the child. Meanwhile, flex is more to like the parent asking the children.
