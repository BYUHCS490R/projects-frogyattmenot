1. What is the root? What is the relationship between folder and file names and the path?

The root is the primary or main link that contains all the projects without any path. For example, http://127.0.0.1:3000. The folder and file names are part of the path. That path starts from the first slash after the root.

2. What is a default document?

It is the main page that you want your user to see. Each page has its own default document. We don't want the users to see our wjhole entire projects like http://127.0.0.1:3000. Instead, to make the main default page, we need to make a new file named index.html.

3. Explain the difference between an absolute link and a relative link. Why would you use relative links?

Absolute link is the link to redirect the users to an external website that is not made by the dev. Relative link is the link that redirects users to another website that is made by the dev. It is usually in the navigation tag. Relative links are usually used to send the users to another page.
