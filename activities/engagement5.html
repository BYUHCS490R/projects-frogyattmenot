2. What is the difference between an ID and a Class?

ID cannot be used more than once. Class can be used like an adjective. Class is reusable. Many programmers tries to avoid using IDs. Can have more than one class for each tag by putting spaces in between two classes.

3. What is the difference between a SPAN tag and a DIV tag?

Div tag is block tag that causes line break while Span tag doesn't. Both dont do changes but helps to select for us to apply the css to it.
