What is the internet?

The internet is defined as a worldwide system of interconnected computer networks
and electronic devices that communicate with each other using an established
set of protocols.

What is the web?

The web is defined as a way to access information to the internet.

What is Git?

Git is a free and open-source distributed version control system (DVCS) designed
to manage and track changes in source code during software development.

What is GitHub?

Github is a cloud-based platform where you can store, share and work together
with others to write a code.

How are we using GitHub?

We’re using GitHub as a platform to store, share, and manage code. It allows
version control with Git, collaboration through branches and pull requests, and
tracking of issues and project progress.

The Meaning Of Each Tags In My Code

1. < !DOCTYPE html >

A declaration that tells the browser that the document is an HTML5 document.

2. < html lang="en" >

The attribute lang="en" specifies the language of the document (English here).

3. < head > and < /head >

Contains metadata about the webpage (not displayed directly on the page).
It usually includes information like title, character encoding, viewport settings,
styles, and links to scripts or CSS.

4. < meta charset="UTF-8" >

Defines the character encoding of the webpage. UTF-8 ensures the page can
display most characters and symbols from different languages.

5. < meta name="viewport" content="width=device-width, initial-scale=1.0" >

Makes the page responsive on mobile devices.
width=device-width tells the
browser to match the screen’s width.
initial-scale=1.0 sets the zoom level when the page first loads.

6. < title > Document < /title >

Sets the title of the webpage, shown in the browser tab.

7. < body > ... < /body >

Contains all the content that will be displayed on the webpage (headings, paragraphs, images, etc.).

8. < h1 > ... < /h1 >

Main heading of a web page. Can be used multiple times in different sections.

9. < h2 > ... < /h2 >

Subeading under < h1 >. Can use multiple < h2 > tags for different sections.

10. < h3 > ... < /h3 >

A subheading under < h2 >.

11. < p > ... < /p >

A paragraph of a text.
