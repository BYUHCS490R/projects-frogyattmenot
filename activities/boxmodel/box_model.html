1. What is the difference between block tags and inline tags?

Block tags are tags that forms a box. They start a new line and take up the full available width. As for inline tags, they do not start a new line and only take up the necessary width for the content. They are mostly used within the block tags for things like styling or marking.

2. What is the margin, border, and padding?

Mergin is the space outside the box. Padding is the space inside the box. Border is the line that goes around the element or the box, which can be invisible or visible.

3. What are units of measurement you can use in CSS?

Pixels is the most common.

4. When would you use a figure tag?

Figure tag is just a block tag that you can put a image in it.
